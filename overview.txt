ArcadeNexus - Online Gaming Website
Category: Web Development

ArcadeNexus is a simple web page where users can play casual games. It runs in any web browser and contains three simple games: Tic-Tac-Toe, Snake, and Memory Match. 

The website has a dashboard that shows user scores and total games played. It saves these scores on the user's computer using the browser's localStorage. The page uses a dark layout with simple cards for the games. It works on both mobile phones and desktop computers.
