ArcadeNexus - Scope of Work
Category: Web Development

1. IN-SCOPE REQUIREMENTS:
- A web page with a header navigation bar, a main hero area, a game selection grid, a user statistics section, gameplay modal popups, and a footer.
- A dark layout using four brand colors (hex codes #0B0C10, #1F2833, #66FCF1, and #45A29E).
- Google Fonts (Orbitron for headers and Inter for body text).
- Three playable games:
  * Tic-Tac-Toe: Player vs computer AI, checks grid cells for wins or draws, and alerts the player.
  * Snake: Plays on an HTML5 canvas element, uses keyboard arrows or WASD, snake grows when eating food, speed increases on every 5 foods, and game ends on boundary or tail crashes.
  * Memory Match: A 4x4 card grid, cards flip face up when clicked, pairs stay visible, mismatches hide after 1 second, and tracks total moves.
- User Stats Dashboard: Displays total games played, scores, wins, losses, and high scores.
- Local Storage: Saves game statistics under the key named "arcadenexus_stats" so data remains after a page reload.
- Reset Button: A button to clear all statistics from local storage and update the counters on the page immediately.
- Global Mute Toggle: Navbar button to turn off and on all background tracks and click/score sound effects.

2. OUT-OF-SCOPE ITEMS (EXCLUDED):
- Multiplayer features using databases or web sockets.
- Server-side backends (Node.js, Express, databases, user account logins).
- Online payments or ads.
- 3D visual graphics or frameworks (Three.js, Phaser).
